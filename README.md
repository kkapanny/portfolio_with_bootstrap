# Портфолио
## Проект
Сайт-портфолио — это визитная карточка в цифровом формате, которая наглядно демонстрирует мои работы, навыки и опыт. Его цель — произвести впечатление на потенциальных клиентов или работодателей и побудить их к сотрудничеству.
